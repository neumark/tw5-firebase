TW5-firebase

This is a Google Firebase-hosted version of TiddlyWiki 5. It uses the FireStore database to save tiddlers, Firebase Auth for signon, Firebase hosting for serving static files and Firebase functions for the tiddler IO API.

Installation

To create your own free instance of ~TW5-firebase, simply fork this repo !! 

{
    "firebaseToken": ...,
    "apikey": ...,
    "refreshToken": ...,
}

firebaseToken can be generated by running:

./node_modules/.bin/firebase login:ci and save the resulting token as .firebase-token

apikey can be downloaded as "Browser Key" on the Gcloud credentials page

refreshToken can be gotten by logging into the wiki, and the running firebase.auth().currentUser.refreshToken in the JS console.

setup

deploy:

npm run deploy

local development:

  1. cd functions; ./run.sh
  2. python -m SimpleHTTPServer
  3. open http://localhost:8000/public/?host=http%3A%2F%2Flocalhost%3A5001%2Fpeterneumark-com%2Feurope-west3%2Fwiki-app%2F
